# Student Performance Analysis Dashboard

## Project Overview
This project analyzes student academic performance using Microsoft Excel.

## Dataset
- Total Students: 649
- Academic performance dataset

## Analysis Performed
- Average Marks Analysis
- Pass/Fail Analysis
- Gender Performance Analysis
- School Performance Comparison

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Dashboard Design

## Key Insights
- Pass Rate: 84.6%
- 549 students passed
- 100 students failed
- GP school achieved higher overall performance than MS
- Female students achieved higher total final grades than male students
